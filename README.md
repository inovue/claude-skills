# claude-skills

inovue's personal collection of [Claude Code](https://docs.anthropic.com/en/docs/claude-code) skills.

## Skills

| Skill | Description |
|-------|-------------|
| [regenerate-claude-md](skills/regenerate-claude-md/SKILL.md) | CLAUDE.mdファイルを現在のコードベースに基づいて再生成する |

## Installation

In Claude Code, run:

```
/plugin marketplace add inovue/claude-skills
/plugin install claude-skills@inovue-claude-skills
```
